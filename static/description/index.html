Point of Sale Session Close

Watch the video

This module allows you to directly close the session, manage the Cash control and print the session summary report from Point of Sale.

Session screen will be displayed as follows.

Manage the Cash control

Click on "Put Money In" button to add some money.

Click on "Take Money Out" button to take money.

Click on "Set Closing Balance" button to add amount in closing cash control.

POS Session Closing Summary

Allows you to print session report with details like Responsible, Point of Sale , Opening Date, Closing Date, Total Sale etc.

POS session summary report in PDF format

Click on "End of session" and "Validate Closing & Posting Entries" button to end the session.
